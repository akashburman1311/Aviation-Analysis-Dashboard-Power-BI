# Aviation-Analysis-Dashboard-Power-BI
An interactive Power BI dashboard that analyzes airline operations, passenger traffic, and load factor performance across multiple carriers, routes, and time periods. The project focuses on transforming raw aviation data into clear, decision-ready insights.

üìå Project Description

This project showcases an interactive Aviation Analysis Dashboard developed using Power BI to analyze airline operations, passenger traffic, and capacity utilization. The goal of this dashboard is to convert large volumes of aviation data into meaningful insights that support data-driven decision-making in the aviation industry.

The dashboard covers key performance indicators such as Total Flights, Total Passengers, Load Factor %, Total Seats, Average Distance, and Average Seats per Flight. It allows users to explore airline performance across routes, carriers, distance ranges, and time periods through dynamic and user-friendly visuals.
